# Task: Update README: run instructions with prerequisites

## Description
Update the `README.md` with comprehensive instructions on how to get the project running locally, including all prerequisites.

## Steps to Complete
1.  Open `README.md`.
2.  Create a "Getting Started" or "Running Locally" section.
3.  List all prerequisites:
    -   .NET SDK (specify version)
    -   Node.js (specify version)
    -   Docker Desktop (for TestContainers)
    -   A code editor (e.g., VS Code, Rider)
4.  Provide step-by-step commands to run the API (`dotnet run`) and the client (`npm install`, `npm run dev`).
5.  Explain how environment variables are managed (e.g., via `appsettings.Development.json`).

## Definition of Done
- [ ] `README.md` has clear, step-by-step instructions for running the project.
- [ ] All software prerequisites are listed.
